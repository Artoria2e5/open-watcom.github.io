Open Watcom V2 Fork

View the Project on GitHub open-watcom/open-watcom

This is unoficial Fork of Open Watcom suite of compilers and tools

Bellow is list of all differences against oficial Open Watcom

  • DOS version of tools now support long file names (LFN) if appropriate LFN driver is loaded by DOS
  • New 2-phase build system, OW can be build by platform native C compiler, no need to use C++ compiler.
  • RDOS compact memory model libraries (device driver) are now fixed due to new build system
  • OW text editor is now self containing, it can be used as standalone tool without any requirements for additional files or configuration

This project is maintained by open-watcom
